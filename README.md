## запуск приложения
скачиваем репозиторий устанавливает node module
нужна андройд студия в ней выбираем проект
и запускаем через npm run android
должно работать

## Запуск Metro Server

```bash
# using npm
npm start

# OR using Yarn
yarn start
```

## Запуск приложения

### Для андроид

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### Для IOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
