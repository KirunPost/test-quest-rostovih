## Запуск Metro Server

```bash
# using npm
npm start

# OR using Yarn
yarn start
```

## Запуск приложения

### Для андроид

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### Для IOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
